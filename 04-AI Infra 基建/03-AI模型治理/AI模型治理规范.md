# AI 模型治理规范

> AI 模型是核心资产——版本管理、安全、偏见检测、监控、合规全链路治理。

---

## 一、模型生命周期

```
数据准备 → 训练 → 评估 → 注册 → 部署 → 监控 → 退役
                                           │
                                     ← 迭代优化
```

---

## 二、模型版本管理

| 要求 | 说明 |
|------|------|
| **模型注册** | 所有模型必须注册到模型仓库（MLflow / 自建） |
| **版本号** | 遵循 `MAJOR.MINOR.PATCH`，与训练数据/代码版本对应 |
| **元数据** | 记录训练数据、超参数、评估指标、框架版本 |
| **不可变** | 已发布的模型权重不可修改，更新 = 新版本 |
| **回滚** | 支持一键回滚到上一版本 |

---

## 三、模型安全

| 风险 | 防护措施 |
|------|----------|
| **Prompt 注入** | 输入过滤、输出检测、权限隔离 |
| **模型窃取** | API 限流、鉴权、模型加密存储 |
| **对抗攻击** | 输入校验、异常检测、鲁棒性测试 |
| **数据泄露** | 训练数据脱敏、推理日志脱敏 |
| **供应链安全** | 基础模型来源验证、SBOM |

---

## 四、模型偏见与公平性

| 检测项 | 方法 | 门槛 |
|--------|------|------|
| 性别偏见 | 测试集偏好分析 | 准确率差异 < 5% |
| 种族偏见 | 不同群体的输出一致性 | 拒绝率差异 < 5% |
| 毒性检测 | 输出内容安全评分 | 毒性评分 < 0.01 |

---

## 五、模型监控

| 指标 | 检测内容 | 告警阈值 |
|------|----------|----------|
| **数据漂移** | 输入分布变化 | PSI > 0.1 |
| **概念漂移** | 预测准确率下降 | 准确率下降 > 5% |
| **延迟** | 推理响应时间 | P99 > 2s |
| **吞吐量** | QPS/TPS | 低于基线 20% |
| **错误率** | 推理失败率 | > 1% |
| **安全事件** | 异常输入/输出 | 任何事件 |

---

## 六、合规要求

| 要求 | 实现 |
|------|------|
| 可解释性 | 关键决策输出解释理由 |
| 数据合规 | 训练数据来源合规，用户数据脱敏 |
| 生成内容标识 | AI 生成内容添加水印/标识 |
| 人工审核 | 高风险场景保留人工审核环节 |
