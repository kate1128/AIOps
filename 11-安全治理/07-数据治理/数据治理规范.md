# 数据治理规范

> 数据是产品的核心资产。数据治理确保数据安全、合规、高质量、可追溯，覆盖数据全生命周期。

---

## 一、治理范围

| 维度 | 说明 |
|------|------|
| **数据分类分级** | 按敏感程度和业务价值对数据分类分级 |
| **数据脱敏** | 非生产环境使用脱敏数据，保护隐私 |
| **数据生命周期** | 从采集到销毁的全流程管理 |
| **数据质量** | 准确性、完整性、一致性、及时性 |
| **数据合规** | 个保法、GDPR 等法规遵从 |

---

## 二、数据分类分级

### 分类维度

| 类别 | 示例 | 说明 |
|------|------|------|
| **用户个人信息** | 姓名、手机号、邮箱、身份证 | 个保法保护对象 |
| **业务数据** | 订单、推理记录、模型输出 | 核心业务资产 |
| **系统数据** | 日志、监控指标、配置 | 运维和排障 |
| **模型数据** | 训练数据、模型权重、Prompt | AI 核心资产 |
| **财务数据** | 合同、License 信息 | 商业敏感 |

### 分级标准

| 等级 | 定义 | 示例 | 管控要求 |
|------|------|------|----------|
| **L4 - 极度敏感** | 泄露造成严重法律/商业风险 | 身份证、密码、密钥、模型权重 | 加密存储、严格审计、最小权限 |
| **L3 - 敏感** | 泄露造成中等风险 | 手机号、邮箱、业务订单 | 脱敏展示、访问审批 |
| **L2 - 内部** | 泄露造成低风险 | 内部文档、组织结构 | 内部可见，不外传 |
| **L1 - 公开** | 无风险 | 产品介绍、帮助文档 | 对外公开 |

## 三、数据脱敏要求

| 数据类型 | 脱敏方式 | 示例 |
|----------|----------|------|
| 手机号 | 中间四位掩码 | 138****1234 |
| 邮箱 | 用户名部分掩码 | u***@example.com |
| 身份证 | 前六后四，中间掩码 | 110101****1234 |
| 姓名 | 保留姓，名掩码 | 张** |
| IP 地址 | 后两段掩码 | 10.0.*.* |
| 密码/密钥 | 不出现在任何日志或数据库 | 哈希存储 |

### 脱敏场景

| 场景 | 要求 |
|------|------|
| 开发/测试环境 | 必须使用脱敏数据，禁止直接复制生产数据 |
| 日志输出 | 自动过滤敏感字段 |
| 数据分析 | 聚合统计可用，明细数据需脱敏 |
| 客户演示 | 使用虚构数据或脱敏数据 |

## 四、数据生命周期

```
采集 → 存储 → 使用 → 传输 → 归档 → 销毁
```

| 阶段 | 管控要求 |
|------|----------|
| **采集** | 最小化采集，明确告知用途，获取用户同意 |
| **存储** | L3/L4 数据加密存储，设置访问权限 |
| **使用** | 最小权限原则，操作审计 |
| **传输** | TLS 加密传输，禁止明文传输敏感数据 |
| **归档** | 定期归档冷数据，设置保留期限 |
| **销毁** | 到期数据安全删除，无法恢复 |

### 数据保留期限

| 数据类型 | 保留期限 | 到期处理 |
|----------|----------|----------|
| 用户账号 | 账号注销后 30 天 | 彻底删除 |
| 业务日志 | 90 天 | 归档后删除 |
| 监控指标 | 180 天 | 归档后删除 |
| 训练数据 | 项目结束后 6 个月 | 脱敏归档或销毁 |
| 财务数据 | 7 年（法律要求）| 加密归档 |

## 五、数据质量 SLA

| 维度 | 定义 | 目标 |
|------|------|------|
| **准确性** | 数据值与真实值的一致程度 | > 99.9% |
| **完整性** | 必填字段的填充率 | > 99.5% |
| **一致性** | 跨系统同字段值的一致性 | > 99% |
| **及时性** | 数据从产生到可用的时间 | 实时 < 1s，离线 < 1h |
| **唯一性** | 无重复记录 | 0 重复 |

---

## 六、数据合规（个保法）

| 要求 | 实现方式 |
|------|----------|
| **告知同意** | 用户协议 + 隐私政策，明确数据用途 |
| **最小必要** | 仅收集业务必需的数据 |
| **数据删除** | 提供账号注销和数据删除功能 |
| **数据可携带** | 支持用户导出自己的数据 |
| **影响评估** | 重要数据处理前进行隐私影响评估(PIA) |
| **跨境传输** | 涉及跨境数据需通过安全评估 |
