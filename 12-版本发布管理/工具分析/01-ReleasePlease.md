# ReleasePlease

> Google 开源的自动化版本发布工具，基于 Conventional Commits 自动生成版本号和 CHANGELOG。

---

## 适用场景

- 需要自动化版本管理和变更日志生成
- 团队已采用 Conventional Commits
- 追求"发布即自动化"的最佳实践

---

## 优劣势

| 优势 | 劣势 |
|------|------|
| Google 维护，社区活跃 | 配置有一定复杂度 |
| GitHub / GitLab 均支持 | 中文 Commit 支持有限 |
| 自动 PR 触发发布 | 需要团队遵循 Commit 规范 |
| 支持 Monorepo | 复杂场景需要自定义配置 |
