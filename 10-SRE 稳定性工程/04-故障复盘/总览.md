# 故障复盘

> Blameless 复盘——找到系统根因并改进，不追究个人责任。

## 复盘流程

```
故障恢复 → 2h 内建复盘文档 → 48h 完成初稿 → 72h 复盘会 → Action Items 跟踪
```

## 故障分级

| 级别 | 影响范围 | 示例 |
|------|---------|------|
| P0 | 全平台不可用 / 数据丢失 | 生产 K8s 控制面宕机 |
| P1 | 核心功能不可用，影响 > 10% 用户 | AI 推理全部失败 |
| P2 | 部分功能降级 | 单个模型不可用 |
| P3 | 轻微影响，有降级方案 | 非核心功能报错 |
