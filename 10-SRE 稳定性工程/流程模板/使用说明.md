# SRE 模板目录

| 文件 | 用途 | 使用频率 |
|---|---|---|
| [01-slo-definition.md](01-slo-definition.md) | 定义服务的 SLI/SLO/Error Budget | 每个核心服务建一份，按月更新 |
| [02-runbook.md](02-runbook.md) | 故障处置手册，每个高频故障场景一份 | 故障发生时查阅，平时补充完善 |
| [03-postmortem.md](03-postmortem.md) | 故障复盘文档 | 每次 P0/P1 故障后 72h 内完成 |
| [04-change-request.md](04-change-request.md) | 变更申请单 | N/H 级变更每次提交 |

## 使用方式

复制对应模板，按格式填写，存放到对应目录：

```
07-sre/
  templates/          # 空白模板（本目录）
  slo/                # 各服务 SLO 定义（按服务名命名）
  runbooks/           # 各故障场景 Runbook
  postmortems/        # 历史故障复盘归档
  changes/            # 变更申请单归档
```
