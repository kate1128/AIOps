## 执行步骤

```bash
# 1. 确认当前版本
kubectl get deployment vllm-qwen7b -n ai-service -o jsonpath='{.spec.template.spec.containers[0].image}'

# 2. 更新镜像
kubectl set image deployment/vllm-qwen7b vllm=vllm/vllm-openai:v0.4.x -n ai-service

# 3. 观察滚动升级
kubectl rollout status deployment/vllm-qwen7b -n ai-service

# 4. 验证（见下方）
```

---

## 回滚方案

**触发条件：** 执行后出现错误率上升 > 1%，或服务不可用

```bash
kubectl rollout undo deployment/vllm-qwen7b -n ai-service
# 预计恢复时间：60s 内
```

---

## 验证步骤

```bash
# 1. Pod 全部 Running
kubectl get pods -n ai-service

# 2. 健康检查
curl http://vllm-service:8000/health

# 3. 观察 Grafana 指标 10 分钟
# - 错误率：vllm:request_success_total
# - TTFT：vllm:time_to_first_token
```
