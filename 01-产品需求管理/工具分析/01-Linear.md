# Linear

> 轻量研发项目管理工具，开发者体验优秀，适合中小团队或追求高效协作的团队。

---

## 适用场景

- 需求管理 + 研发任务追踪一体化
- 追求简洁高效的项目管理流程
- 不需要复杂的自定义工作流和权限体系

---

## 优劣势

| 优势 | 劣势 |
|------|------|
| 用户体验极佳，上手快 | 国内访问可能不稳定 |
| 快捷键驱动，操作效率高 | 缺少中文界面 |
| Cycles 机制天然适合迭代管理 | 企业级报表能力弱于 Jira |
| Roadmap 视图清晰直观 | 权限模型相对简单 |
| API 完善，自动化能力强 | 无原生飞书/企业微信集成 |

---

## 与 Jira 对比

| 维度 | Linear | Jira |
|------|--------|------|
| 用户学习成本 | 低 | 中高 |
| 自定义程度 | 中 | 高 |
| 企业报表 | 基础 | 强大 |
| 三方集成 | 丰富但中国生态弱 | 全面 |
| 价格 | 按用户 | 按用户 |
