# Jira

> 企业级项目管理平台，功能全面、可定制性高，适合中大型团队或需要复杂工作流的场景。

---

## 适用场景

- 需要精细化的权限管控和工作流
- 复杂的企业级报表和仪表盘
- 多团队、多项目协同管理
- 需要和飞书/企业微信深度集成

---

## 优劣势

| 优势 | 劣势 |
|------|------|
| 工作流自定义能力极强 | 学习成本高，新用户上手慢 |
| 报表和仪表盘丰富 | 配置复杂，维护成本高 |
| 权限模型精细 | 流畅度不如 Linear |
| 插件生态丰富 | 自建服务器版本维护重 |
| 支持 Scrum/Kanban 全模式 | 价格随用户数增长快 |
