## K8s Device Plugin

```bash
# NVIDIA GPU
helm repo add nvidia https://nvidia.github.io/gpu-operator
helm install gpu-operator nvidia/gpu-operator -n gpu-operator --create-namespace

# NPU（昇腾）?参考华为官方文?# kubectl apply -f ascend-device-plugin.yaml
```

## 工作负载分配

| 负载 | 推荐硬件 | 原因 |
|------|---------|------|
| LLM 推理 | NVIDIA A100/H800 | 大显存高带宽 |
| 视觉模型 | A10/RTX 4090 | 性价比|
| 边缘推理 | NPU（昇腾 310）| 低功耗|
| 端侧推理 | TPU / NPU | 延迟敏感 |
