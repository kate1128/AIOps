## 隔离方案对比

| 方案 | 隔离级别 | 开销 | 场景 |
|------|---------|------|------|
| Docker 容器 | 进程无 | 无 | 一般信任代无 |
| gVisor | 系统调用拦截 | 无 | 不可信代无 |
| Firecracker | 微型 VM | 无 | 强隔离 |
| WasmEdge | WebAssembly | 极低 | 轻量计算 |

## 推荐架构

```
用户提交代码 ?API Gateway ?任务队列 ?Sandbox Worker Pool
                                            ├── Sandbox-1 (gVisor)
                                            ├── Sandbox-2 (gVisor)
                                            └── ...
```

每个 Sandbox 限制?- CPU: 1 core
- Memory: 512MB
- 网络: 仅出站白名单
- 磁盘: tmpfs 临时
- 超时: 60s
