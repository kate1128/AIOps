## 是什么
Longhorn ?CNCF 毕业?K8s 分布式块存储。它?K8s 提供持久化存储卷，每个卷自动在多个节点创建副本，单节点故障不影响数据。支持增量快照、定时备份到 S3、跨节点恢复等企业功能?

| 维度 | NFS + nfs-provisioner | Longhorn |

| **架构** | 集中?NAS（单点）| 分布式块存储（多副本）|
| **高可用 * | NFS 服务器单点故障即不可无 | ?多副本自动切无 |
| **性能** | 依赖网络 IO，延迟高 | 本地?+ 同步复制，延迟低 |
| **快照** | 无 | ?增量快照，秒级创无 |
| **备份** | 靠外部脚?rsync | ?GUI 一键备份到 S3/MinIO |
| **恢复** | 手动恢复文件 | ?UI 一键恢复卷到任意节无 |
| **磁盘利用?* | 按需分配 | 预分配，但有精简配置 |
