# Snyk — 开发者安全平台

> 与 Trivy 的对比参考
> 当前使用：无（计划引入 Trivy）

---

## 是什么

Snyk 是商业化的开发者安全平台，提供代码扫描（SAST）、开源依赖扫描（SCA）、容器镜像扫描、基础设施即代码（IaC）扫描。与 Trivy 是单一 CLI 工具不同，Snyk 是全栈 SaaS 平台。

---

## 与 Trivy 的核心区别

| 维度 | Trivy | Snyk |
|------|-------|------|
| **模式** | 开源 CLI 工具 | SaaS 平台 |
| **扫描范围** | 镜像/FS/Git/K8s | 代码 + 依赖 + 镜像 + IaC |
| **修复建议** | CVE 信息 | ✅ 自动升级版本 + PR 修复 |
| **策略引擎** | 无 | ✅ 自定义合规策略 |
| **仪表盘** | 无 | ✅ 团队 Dashboard + 报告 |
| **定价** | 免费 | 免费额度有限，团队版收费 |
| **CI 集成** | 脚本 | 原生 + CLI |

---

## 引入 Snyk 你能得到什么

| 收益 | 说明 |
|------|------|
| ✅ 全栈安全 | 代码 + 依赖 + 容器 + IaC 一套平台覆盖 |
| ✅ 自动修复 PR | Snyk 自动创建 PR 升级有漏洞的依赖 |
| ✅ 策略管理 | 自定义合规规则，不符合规则的阻断流水线 |
| ✅ 团队 Dashboard | 安全态势一目了然，老板汇报利器 |

## 引入 Snyk 的代价

| 代价 | 说明 |
|------|------|
| ❌ 收费 | 团队版按开发者收费，成本不低 |
| ❌ 数据上云 | Snyk 是 SaaS，需要将 SBOM/依赖信息发送到 Snyk 云 |
| ❌ 超范围 | 如果只需要镜像扫描，Snyk 功能过剩 |
