# 基础指标

> 节点、网络、磁盘、硬件、容器、拨测——系统跑在什么上面，它们是否健康。

---

## 覆盖范围

| 能力 | 工具 | 状态 | 说明 |
|------|------|------|------|
| 主机 CPU/内存/磁盘 | node-exporter + Zabbix | 🟡 部分覆盖 | K8s 节点有 node-exporter，非 K8s 节点由 Zabbix |
| 网络设备 | Zabbix SNMP | 🟡 部分覆盖 | 部分设备已采集 |
| 磁盘/存储 | Zabbix + Exporter | 🟡 部分覆盖 | 缺 MinIO/NFS 指标 |
| 容器指标 | cAdvisor | 🔴 未覆盖 | 未部署到 Docker 宿主机 |
| 硬件传感器 | IPMI Exporter | 🔴 未覆盖 | 未部署 |
| 证书过期 | Blackbox / Gatus | 🔴 未覆盖 | 未配置 |
| 外部 HTTP 拨测 | Gatus / UptimeKuma | 🔴 未覆盖 | 无从外部检测可用性 |
| TCP 端口探活 | Zabbix + Blackbox | 🟡 部分 | Zabbix 有部分检查 |

## 覆盖矩阵（最终目标）

| 节点类型 | 指标采集 | 采集方式 |
|---------|---------|---------|
| K8s 节点 | node-exporter + kubelet | DaemonSet |
| Docker 宿主机 | node-exporter + cAdvisor + Docker metrics | systemd / 容器 |
| 二进制服务节点 | node-exporter | systemd |
| GPU 节点 | + DCGM Exporter + nvidia-gpu-exporter | DaemonSet/systemd |
| 网络设备 | snmp_exporter | 容器 / systemd |
| 所有节点 | IPMI Exporter（有 BMC 时） | systemd |
