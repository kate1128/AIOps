# 供应商与第三方管理规范

> 供应商和第三方组件是供应链安全的关键环节——评估、监控、审计全流程管理。

---

## 一、供应商管理范围

| 类别 | 示例 | 管理重点 |
|------|------|----------|
| **云服务商** | 阿里云、华为云 | SLA、安全资质、数据主权 |
| **软件供应商** | 数据库、中间件厂商 | License 合规、漏洞响应 |
| **第三方组件** | 开源库、Docker 镜像 | CVE 扫描、License 合规 |
| **外包开发** | 外包团队 | 代码审计、交付质量 |
| **硬件供应商** | 服务器、GPU 硬件 | 供应稳定性、认证合规 |

---

## 二、供应商评估流程

```
需求确认
  → 候选供应商筛选
  → 安全资质审查（等保/ISO）
  → 技术方案评估
  → 商务谈判
  → 合同签署（含 SLA 和安全条款）
  → 接入实施
  → 持续监控
  → 定期复审（年度）
```

---

## 三、供应商分级

| 等级 | 定义 | 审查频率 | 要求 |
|------|------|----------|------|
| **T1 - 核心** | 直接影响核心业务 | 年度审计 | 必须通过安全审查、有备选方案 |
| **T2 - 重要** | 支撑业务运行 | 年度问卷 | 安全问卷合规 |
| **T3 - 一般** | 辅助工具 | 合同审查 | 标准条款 |

---

## 四、第三方组件管理

| 管控项 | 要求 |
|--------|------|
| **SBOM** | 所有产品必须生成 SBOM |
| **CVE 扫描** | 每次构建自动扫描，Critical 阻断 |
| **License 合规** | 禁止 GPL 等传染性 License 引入商业产品 |
| **版本锁定** | 生产环境锁定所有依赖版本 |
| **废弃组件** | 已知废弃组件必须在 30 天内替换 |
