## 推荐引入

1. **补齐 Promtail 采集范围**：三种部署形态全覆盖
   - K8s Pod 日志 → Promtail DaemonSet（已有）
   - Docker 容器日志 → 宿主机 Promtail 读 `/var/lib/docker/containers/*/*-json.log`
   - 二进制 Java 日志 → 宿主机 Promtail 读 `/opt/smartvision/logs/**/*.log`
2. **日志格式规范**：所有服务输出 JSON 格式日志，包含 `timestamp`、`level`、`service`、`traceId`、`message`
3. **日志告警**：LogQL 条件触发，如 `rate({service="java"}|= "ERROR"[5m]) > 10`
